Info

This is the page to get info about the Appel Workshop.

Rules

This level allows you to create levels and play levels. Therefore, there need to be rules for both uploading and verifying levels.

Uploading Levels

  • All levels you upload must either be created by you, or have credit to the original creator in the description.
  • All levels uploaded must either have a valid code or a Scratch project link to an Appel mod.
  • All levels must be theoretically possible, or it will get removed.
  • All levels uploaded must not have swearing or other inappropriate words inside the title, description, or level.

Verifying levels

  • All levels must be verified in real-time.
  • All levels must be verified at 30FPS, or as close as possible. Levels cannot be verified in less than 25FPS.
  • All level verifications must be played by you.
  • Level verifications must require either a replay code or a video. Be sure to keep both - if there are any suspicions about your run, we may ask for one or the other.
  • Level verifications must be played in either the original mod or an allowed mod for use. Replay codes must work in this mod.
